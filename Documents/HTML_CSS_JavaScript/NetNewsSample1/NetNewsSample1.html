飛車山社サイバー攻撃、復旧の目途は

飛車山社がサイバー攻撃を受けてからn日。ニヤニヤ動画・生放送が使えなくなっているほか、同社が経営するネットショッピングサイトやアニメ番組のサイトが使えないなど、現在まで多大なる影響を与えている。

記録が途切れたことも

飛車山社は新たなサーバー構築を行っている。現サーバーはシャットダウンされており、動画投稿や生放送機能が使えなくなっているほか、16日までの公式放送を中止が発表されている。

サイバー攻撃で動画が投稿できないことから、17年間続いた「毎日ボカロオリジナル曲が投稿された」という記録が途切れることに。ファンは悲しみに暮れた。
